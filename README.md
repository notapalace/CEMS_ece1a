# CEMS: Campus Event Management System 

**CEMS** is an integrated event management solution designed to streamline the academic event lifecycle. By providing distinct interfaces for students and organizers, the system automates registration, attendance tracking, and post-event evaluation.

## Features: Students' Perspective
The mobile-oriented interface focuses on a seamless user journey from discovery to reward.

* *A multi-step signup process requiring personal information and organization selection.*
* *Users create unique credentials to ensure data privacy before browsing events.*
* *A navigation-friendly dashboard where students can "Join" events.*
* *View comprehensive event details (date, status, location) and confirm attendance via email and preferred modality.*
* *Automated generation of a QR-code ticket for event check-in.*
* *Post-event evaluation forms that reward users with digital badges and profile updates.*

## Features: Organizers' Perspective
The web-based administrative dashboard provides full control over campus activities.

* *A centralized form to define modality, timing (start/end), and description for new events.*
* *Real-time tools to implement, edit, and track attendance*
* *Manual override to add participants, verify evaluation status, and distribute badges.*


## Technical Workflow
* *Data captured during signup is automatically mapped and reflected in the user's Profile frame (Student ID, Name).*
* *Tickets are rendered on a canvas representing unique QR identifiers for entrance status.*
* *The system locks badge rewards behind an evaluation form to ensure high-quality feedback for organizers.*
